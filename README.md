# 🦞 Qt Modbus 调试工具 v1.1 - RS485 串口版

一个基于 Qt6 的 Modbus RTU (RS485) 调试工具，支持帧超时检测、自动 CRC 校验、循环发送等功能。

## ✨ 功能特性

### 核心功能
- 🔌 **RS485 串口通信** - 支持自定义波特率、数据位、停止位、校验位
- 📡 **Modbus RTU 协议** - 支持功能码 03/04/06/16
- ⏱️ **帧超时检测** - 可配置超时时间，自动区分数据帧
- 🔐 **自动 CRC 校验** - Modbus RTU 标准 CRC-16，发送自动附加，接收自动验证

### 日志系统
- 📤 **发送日志** - 显示所有发送数据，统计次数和字节数
- 📥 **接收日志** - 显示所有接收数据，CRC 校验状态标记
- 📊 **合并日志** - 时间轴方式显示收发全过程
- 💾 **日志导出** - 支持导出为 TXT 文件
- ⏱️ **毫秒时间戳** - 精确到毫秒的时间记录

### 循环发送
- 🔄 **多任务并发** - 支持多个发送任务同时执行
- ⏲️ **可配置间隔** - 每个任务独立设置发送间隔
- 🔢 **次数控制** - 支持无限循环或指定次数
- ▶️ **启停控制** - 启动、暂停、停止灵活控制

### 数据格式
- 🔢 **十六进制显示** - 所有数据以十六进制格式显示
- 📝 **自定义发送** - 支持手动输入任意十六进制数据

## 🛠️ 本地编译 (Windows)

### 环境要求

- Windows 10/11
- Qt 6.6+ (包含 Qt SerialBus 模块)

```powershell

# 3. 部署 (复制依赖)
cd build\Release
windeployqt QtModbusTool.exe --release
```

## 📖 使用说明

### 1. 串口连接

1. 选择串口号（如 COM1）
2. 设置波特率（默认 9600）
3. 设置数据位、停止位、校验位
4. 设置帧超时时间（默认 50ms）
5. 勾选"自动 CRC"（发送时自动附加 CRC）
6. 点击"🔌 打开串口"

### 2. 手动发送

**读取寄存器：**
- 选择功能码（03 或 04）
- 设置从站 ID、起始地址、数量
- 点击"📖 读取"

**写入寄存器：**
- 选择功能码（06 或 16）
- 设置从站 ID、起始地址
- 输入写入值（逗号分隔）
- 点击"✏️ 写入"

**自定义发送：**
- 在自定义框输入十六进制数据（如 `01 03 00 00 00 0A`）
- 点击"📤 发送"
- 如果勾选"自动 CRC"，会自动附加 CRC 校验码

### 3. 循环发送

1. 在自定义发送框输入数据
2. 点击"+ 添加任务"
3. 在表格中设置间隔时间和发送次数
4. 勾选"启用"复选框
5. 点击"▶ 启动循环"

### 4. 日志管理

- **查看日志**：三个日志面板分别显示发送、接收、合并日志
- **导出日志**：点击每个面板的"💾 导出"按钮
- **清空日志**：点击"🗑 清空"按钮
