# 🦞 Qt Modbus 调试工具

一个基于 Qt6 的 Modbus TCP 调试工具，支持读取/写入寄存器、设备扫描等功能。

## ✨ 功能特性

- 📡 **Modbus TCP 连接** - 支持自定义服务器地址、端口、从站 ID
- 📖 **读取操作** - 支持功能码 03(读保持寄存器)、04(读输入寄存器)
- ✏️ **写入操作** - 支持功能码 06(写单个寄存器)、16(写多个寄存器)
- 🔍 **设备扫描** - 快速检测设备响应
- 📊 **数据表格** - 直观显示寄存器数据
- 📝 **实时日志** - 记录所有操作和错误信息

## 🛠️ 本地编译 (Windows)

### 环境要求

- Windows 10/11
- Qt 6.6+ (包含 Qt SerialBus 模块)
- CMake 3.16+
- Visual Studio 2019 或更高

### 编译步骤

```powershell
# 1. 配置
cmake -S . -B build -G "Visual Studio 16 2019" -A x64

# 2. 编译
cmake --build build --config Release

# 3. 部署 (复制依赖)
cd build\Release
windeployqt QtModbusTool.exe --release
```

## 📖 使用说明

1. **连接设备**
   - 输入 Modbus TCP 服务器地址（如 192.168.1.100）
   - 端口默认 502
   - 从站 ID 默认 1
   - 点击"连接"

2. **读取数据**
   - 选择功能码（03 或 04）
   - 设置起始地址和数量
   - 点击"读取"

3. **写入数据**
   - 选择功能码（06 或 16）
   - 设置起始地址
   - 输入值（多个值用逗号分隔）
   - 点击"写入"
